# Kafka in Docker
this is a collection of scripts that helps set up an Apache Kafka cluster also with a user interface that helps visualise partitions, topics and a few other parameters

It is based upon
 * https://github.com/wurstmeister/kafka-docker; https://hub.docker.com/r/wurstmeister/kafka/
 * https://hub.docker.com/r/wurstmeister/zookeeper/
 * https://hub.docker.com/r/nodefluent/kafka-rest/
 * https://hub.docker.com/r/nodefluent/kafka-rest-ui/
